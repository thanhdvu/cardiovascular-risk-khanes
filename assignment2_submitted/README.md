## 프로젝트 소개
KNHANES 2017–2023 데이터를 활용하여 COVID-19 전후 한국 청년층(19–39세)의 고혈압 위험을 예측하는 머신러닝 프로젝트입니다.

## 목표
* COVID 전후 건강 변화 분석
* 고혈압 여부 예측
* 머신러닝 모델 성능 비교 (AUC 기준)

## 프로젝트 데이터 스크리닝 과정
1. 데이터 전처리
2. EDA
3. PyCaret 모델 비교
4. Top 3 모델 선정
5. ROC Curve / Confusion Matrix / Feature Importance 분석

## 사용 모델
* Logistic Regression
* Random Forest
* XGBoost
* LightGBM
